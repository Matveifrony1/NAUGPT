Заголовок: Для викладачів та студентів НАУ проведено вебінар щодо нових можливості від компанії Microsoft
Дата: 13/03/2023
URL: https://nau.edu.ua/ua/news/2023/3/dlya-vikladachiv-ta-studentiv-nau-provedeno-vebinar-shchodo-novih-mozhlivosti-vid-kompanii-microsoft.html
==================================================

Нещодавно на Факультеті кібербезпеки та програмної інженерії відбулася онлайн зустріч з випускником НАУ Іваном Азаровим, який провів ознайомче заняття для викладачів та студентів НАУ щодо безкоштовного отримання Microsoft GitHub Student Pack.
Студентський пакет GitHub включає доступ до понад 100 інструментів і ресурсів, які допоможуть нашим викладачам та студентам навчитися кодувати, вдосконалювати свої навички та створювати проекти.
Користувачі, серед іншого, отримають такі інструменти:
Azure Microsoft і DigitalOcean: ознайомлення з хмарним середовищем, віртуальні машини на базі Windows Linux та з твердотільними накопичувачами;
GitHub Pro: пакет інструментів, які допомагають співпрацювати з іншими, керувати своїм кодом і створювати проекти програмного забезпечення;
Namecheap,
Name.com
: реєстратори доменних імен, який може допомогти зареєструвати доменне ім’я для вашого проекту;
JetBrains: доступ до професійної версії IntelliJ IDEA, одного з найпопулярніших інтегрованих середовищ розробки (IDE) для розробки Java, C++, Javascript, Python.
