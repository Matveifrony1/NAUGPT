Заголовок: Співпраця НН ІКІТ з радою з конкурентноспроможності індустрії ІКТ
Дата: 25/01/2016
URL: https://nau.edu.ua/ua/news/2016/1/spivpraczya-nn-ikit-z-radoyu-z-konkurentnospromozhnosti-industrii-ikt.html
==================================================

Навчально-науковий інститут Комп’ютерних інформаційних технологій НАУ із січня 2016 року розпочав співпрацю із Радою з конкурентоспроможності індустрії ІКТ щодо реалізації проекту «Європейські освітні ініціативи».
Програма проекту передбачає надання студентам та абітурієнтам ІТ-спеціальностей грантів на навчання сучасним інформаційним технологіям в комерційних навчальних центрах, а також виплату кращим студентам іменних стипендій.
За результатами навчання студенти зможуть отримати затребувані на ринку ІТ-спеціальності:
• C # .NET Developer;
• JAVA Developer;
• Frontend Developer;
• Full stack Developer (Frontend + Backend);
• Game Developer;
• PHP Developer.
На отримання гранту можуть претендувати як абітурієнти, що планують навчатися в Національному авіаційному університеті, так і студенти НАУ.
